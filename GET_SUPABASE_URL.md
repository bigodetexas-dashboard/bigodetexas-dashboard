# Como Pegar a URL Correta do Supabase

## Passo 1

Acesse: <https://supabase.com/dashboard/project/uvyhpedcgmroddvkngdl/settings/database>

## Passo 2

Role a página até encontrar **"Connection string"** ou **"URI"**

## Passo 3

Copie a URL que aparece (deve começar com `postgresql://postgres...`)

## Passo 4

Cole a URL aqui para eu configurar!

---

**Nota:** A URL deve ter sua senha (`Lissy@2000`) substituída por `Lissy%402000`
